12V Lineaire Elektrische Actuator
Na vergelijking van alle opties is de elektrische lineaire actuator (750N / 75kg kracht, 50mm slag) de absolute beste keuze. Geen lekkagerisico, geen pomp, direct aanstuurbaar via Arduino L298N driver, ingebouwde rem houdt positie vast bij stroomverlies, en beschikbaar voor ~€30–40 op AliExpress.

Hoe werkt het? De lineaire actuator zit parallel aan je elleboogscharnier. Als jij je bicep aanspant → EMG-sensor detecteert dit → Arduino opent de L298N driver → actuator strekt uit → elleboog buigt krachtig → jij tilt makkelijk tot 75kg mee. Loslaten = actuator trekt terug. De ingebouwde rem houdt de positie vast als stroom stopt.

// ╔════════════════════════════════════════╗
// ║  NEXO-1 // Elektrisch Arm Exoskelet   ║
// ║  Arduino Nano + EMG Sensor + L298N    ║
// ║  Lineaire actuator 750N / 75 kg       ║
// ╚════════════════════════════════════════╝

// ── PIN DEFINITIE ─────────────────────────
#define EMG_PIN      A0   // EMG sensor uitgang
#define ACT_IN1      7    // L298N IN1 → actuator OMHOOG
#define ACT_IN2      8    // L298N IN2 → actuator OMLAAG
#define ACT_ENA      9    // L298N ENA (PWM snelheidscontrole)
#define NOODSTOP     2    // NC knop op pols (LOW = STOP ALLES)
#define LED_STATUS   13   // Ingebouwde LED voor status

// ── KALIBRATIE (aanpassen per persoon!) ───
int DREMPEL_AAN   = 420;  // EMG waarde: spier gespannen
int DREMPEL_UIT   = 180;  // EMG waarde: spier ontspannen
int SNELHEID      = 220;  // PWM 0–255 (hoger = sneller maar minder kracht)
int VERTR_OMLAAG  = 350;  // ms: hoe lang omlaag bewegen bij ontspanning
int LOOP_DELAY    = 35;   // Reactievertraging in ms

// ── SETUP ─────────────────────────────────
void setup() {
  Serial.begin(9600);
  pinMode(ACT_IN1,    OUTPUT);
  pinMode(ACT_IN2,    OUTPUT);
  pinMode(ACT_ENA,    OUTPUT);
  pinMode(LED_STATUS, OUTPUT);
  pinMode(NOODSTOP,   INPUT_PULLUP);
  allesUit();
  Serial.println("NEXO-1 ONLINE // Wacht op kalibratie...");
}

// ── HOOFD LOOP ────────────────────────────
void loop() {

  // 1. NOODSTOP CHECK (altijd eerst!)
  if (digitalRead(NOODSTOP) == LOW) {
    allesUit();
    digitalWrite(LED_STATUS, HIGH);
    Serial.println("!! NOODSTOP GEACTIVEERD -- herstart vereist !!");
    while(true) { delay(100); } // volledig halt
  }

  // 2. EMG SENSOR UITLEZEN
  int emg = analogRead(EMG_PIN);
  Serial.print("EMG: "); Serial.println(emg);

  // 3. ACTUATOR AANSTURING LOGICA
  if (emg > DREMPEL_AAN) {
    // ▲ Spier GESPANNEN → arm OMHOOG
    analogWrite(ACT_ENA,  SNELHEID);
    digitalWrite(ACT_IN1, HIGH);
    digitalWrite(ACT_IN2, LOW);
    digitalWrite(LED_STATUS, HIGH);

  } else if (emg < DREMPEL_UIT) {
    // ▼ Spier ONTSPANNEN → arm omlaag (gecontroleerd)
    analogWrite(ACT_ENA,  160); // langzamer omlaag
    digitalWrite(ACT_IN1, LOW);
    digitalWrite(ACT_IN2, HIGH);
    delay(VERTR_OMLAAG);
    allesUit(); // rem actief: houdt positie vast
    digitalWrite(LED_STATUS, LOW);

  } else {
    // = Neutrale zone → HOUD POSITIE (actuator rem)
    allesUit();
  }

  delay(LOOP_DELAY);
}

// ── HELPER: alles uitschakelen ────────────
void allesUit() {
  digitalWrite(ACT_IN1, LOW);
  digitalWrite(ACT_IN2, LOW);
  analogWrite(ACT_ENA,  0);
}
🎯 Kalibratie: EMG Drempelwaarden Instellen
1
Upload code → open Serial Monitor in IDE → stel in op 9600 baud
2
Ontspan bicep volledig → lees EMG-waarde meerdere keren → dit is je rust-waarde
3
Span bicep stevig aan → lees EMG-waarde → dit is je actief-waarde
4
Stel in de code: DREMPEL_AAN = (rust + actief) ÷ 2 · DREMPEL_UIT = rust + 60
5
Pas SNELHEID aan: lager = meer kracht/minder snelheid. Begin bij 180, verhoog naar 255 indien nodig. Actuator mag niet oververhitten!
Pro tip: Plak de EMG elektroden altijd op dezelfde plek (markeer met een stift de huidposities). Elektroden worden minder goed na 2–3 sessies -- vervang ze regelmatig voor betrouwbaar signaal.
